Lesson 7 || intro to web10 + web10 hello world

Make a Web10 Account + Identity
  1. go to the web10 app store, and click on web10 social.
  2. hit the login button
  3. if you haven't signed up for web10, sign up for web10 + verify your phone number!
  4. go to the my bio page, change the photo, bio, and name
  5. add some other people in the class as contacts.
  6. congratulations, you made a web10 decentralized identity / account

Make Your First web10 app!
  1. make a new repl html, css, js project
  2. copy all of the lesson 7 files into the repl
  3. in script.js, set the web10_authenticator variable to your orgs auth server
  4. set the web10_registration variable orgs registration server
  5. successfully log into the hello world app through web10.
  6. Go to your orgs web10 app server in the web browser.
  7. Try changing the name of your app in the manifest.json to something unique.
  8. Look at the organization app store again. Congrats on publishing your first web10 app!
  9. on repl, hit the github branch icon , commit, and push your changes to a new repo named lesson7.
